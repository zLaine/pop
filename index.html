Pop

  • High Concept: Pop all the bubbles to win.
  • Player Experience Goals: Players will have fun trying to beat their best times.
  • Genre: An abstract puzzle game. (Sort of?)
  • Game play: Use the mouse to click on the bubbles to pop them.
  • Prototype goal: Experimentation an abstract game, as well as using game states.
  • Features: This game feautures a very satisfying popping sound and entertaining gameplay.
  • Setting/Premise: This game is abstract, so there isn't really one.
  • Target Audience: People who are entertained by popping bubble wrap -- so everyone!
  • Play time: Varies on how well you do, anywhere from 20 seconds to a minute usually.
  • Strategies: Try popping all in one area and then moving on to other high density areas before chasing single bubbles down.
  • Prototype Assets: All assets made by me.
  • Prototype Closest Other Game: I was inspired by Zack Fergus' slime abstract game from assignment 7. I'm sure there are also pre-existing versions of this game.
